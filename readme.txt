== Description ==

This WordPress plugin automatically inserts boo-box links in the images in your blog using alt and title tags as search terms for products.

= How does it work? =

1. When you are ready to insert an image in an article, remember to fill the alt or title tag for the image. The plugin automatically inserts the boo-box link in the image. When the user clicks on the content, boo-box will display the related products.

== Installation ==

1. Copy the content of the archive into the WordPress plugin folder (/wp-content/plugins).
1. Login to the administration page at WordPress, go into the Plugins page and activate "image-boobox". Still on the plugins page, select the "boo-box configuration" option and fill out the required fields under affiliate programs.
